# 🎵 Java Console Audio Player

This is a simple Java project that plays audio files using the `Clip` class from the `javax.sound.sampled` package.

The program runs in the console and lets the user control playback with basic commands.

---

## 🚀 Features

- Play audio ▶️
- Stop audio ⏹️
- Reset audio 🔄
- Quit program ❌
- Handles common audio errors

---

## 🛠️ Technologies

- Java
- javax.sound.sampled (Audio API)
- Scanner (user input)
